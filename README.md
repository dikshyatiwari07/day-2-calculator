# Day 2 - Calculator 🧮

This is a simple calculator built using Python.

## 📌 Features
- Performs Addition (+)
- Performs Subtraction (-)
- Performs Multiplication (*)
- Performs Division (/)
- Runs in a loop for multiple calculations
- Handles invalid input using try-except

## ▶️ How to Run
1. Open terminal
2. Navigate to the project folder
3. Run:
   python day2_calculator.py

## 🧠 What I Learned
- How to use loops (while)
- Taking user input in Python
- Using if-elif-else conditions
- Handling errors using try-except
- Writing cleaner and structured code
