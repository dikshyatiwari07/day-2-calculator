# Day 2 - Calculator 🧮

This is a simple calculator built using Python.

## 📌 Features
- Performs Addition (+)
- Performs Subtraction (-)
- Performs Multiplication (*)
- Performs Division (/)
- Runs in a loop for multiple calculations
- Handles invalid input using try-except

## ▶️ How to Run
1. Open terminal
2. Navigate to the project folder
3. Run the command:
   python day2_calculator.py

## 🧠 What I Learned
- Using loops (while)
- Taking user input
- Conditional statements (if-elif-else)
- Error handling in Python
